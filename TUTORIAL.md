## 1. モード

Neovim にはいくつかのモードがあります。

| モード | 説明 | 移行方法 |
|---|---|---|
| **Normal** | 移動・操作の基本モード | `Esc` |
| **Insert** | テキスト入力 | `i` / `a` / `o` |
| **Visual** | テキスト選択 | `v`（文字）/ `V`（行）/ `<C-v>`（矩形） |
| **Command** | コマンド実行 | `:` |

---

## 2. 基本操作

### カーソル移動

| キー | 動作 |
|---|---|
| `h / j / k / l` | 左 / 下 / 上 / 右 |
| `w / b` | 次の単語の先頭 / 前の単語の先頭 |
| `e` | 次の単語の末尾 |
| `0 / $` | 行頭 / 行末 |
| `gg / G` | ファイル先頭 / ファイル末尾 |
| `<C-d> / <C-u>` | 半画面下 / 上にスクロール |
| `{行番号}G` | 指定行へジャンプ（例: `10G`） |

### 編集

| キー | 動作 |
|---|---|
| `i / I` | カーソル前 / 行頭から Insert モード |
| `a / A` | カーソル後 / 行末から Insert モード |
| `o / O` | 下に新行 / 上に新行を追加して Insert モード |
| `x` | カーソル下の文字を削除 |
| `dd` | 行を削除（カット） |
| `yy` | 行をコピー |
| `p / P` | カーソルの後 / 前に貼り付け |
| `u` | アンドゥ |
| `<C-r>` | リドゥ |
| `ciw` | 単語を削除して Insert モード |
| `di"` | `"` 内のテキストを削除 |
| `.` | 直前の操作を繰り返す |

### Visual モードでの操作

| キー | 動作 |
|---|---|
| `>` / `<` | インデントを増やす / 減らす（選択維持） |
| `J` / `K` | 選択行を下 / 上に移動 |
| `y` | コピー |
| `d` | 削除 |
| `p` | 貼り付け（元の内容を上書きしない） |

## 3. ファイル操作

| キー | 動作 |
|---|---|
| `<leader>w` | 保存 |
| `<leader>q` | 終了 |
| `<leader>Q` | 全ウィンドウ終了 |
| `:w` | 保存 |
| `:q` | 終了 |
| `:wq` | 保存して終了 |
| `:q!` | 変更を破棄して終了 |
| `:e {ファイル名}` | ファイルを開く |

---

## 4. ウィンドウ・バッファ

### ウィンドウ分割

| キー | 動作 |
|---|---|
| `:sp` | 水平分割 |
| `:vsp` | 垂直分割 |
| `<C-h/j/k/l>` | ウィンドウ間を移動 |
| `:close` | 現在のウィンドウを閉じる |

### バッファ操作

| キー | 動作 |
|---|---|
| `<S-h>` | 前のバッファへ |
| `<S-l>` | 次のバッファへ |
| `<leader>bd` | 現在のバッファを閉じる |
| `:ls` | バッファ一覧 |

## 5. 検索・置換

| キー | 動作 |
|---|---|
| `/{パターン}` | 前方検索 |
| `?{パターン}` | 後方検索 |
| `n / N` | 次の / 前のマッチへ |
| `<Esc>` | 検索ハイライト解除 |
| `:%s/old/new/g` | ファイル全体で置換 |
| `:%s/old/new/gc` | 確認しながら置換 |

## 6. ファイルエクスプローラー (neo-tree)

| キー | 動作 |
|---|---|
| `<leader>e` | エクスプローラーの開閉 |

エクスプローラー内のキー操作:

| キー | 動作 |
|---|---|
| `Enter` | ファイルを開く / ディレクトリを展開 |
| `a` | ファイル / ディレクトリを作成 |
| `d` | 削除 |
| `r` | リネーム |
| `y` | コピー |
| `x` | カット |
| `p` | 貼り付け |
| `q` | エクスプローラーを閉じる |

## 7. ファジーファインダー (Telescope)

| キー | 動作 |
|---|---|
| `<leader>ff` | ファイル名で検索 |
| `<leader>fg` | ファイル内容をgrep検索 |
| `<leader>fb` | 開いているバッファ一覧 |
| `<leader>fh` | ヘルプタグ検索 |

Telescope ウィンドウ内のキー操作:

| キー | 動作 |
|---|---|
| `<C-n> / <C-p>` | 次 / 前の候補に移動 |
| `Enter` | 選択して開く |
| `<C-x>` | 水平分割で開く |
| `<C-v>` | 垂直分割で開く |
| `<Esc>` | 閉じる |

## 8. LSP (コード補助)

LSP（Language Server Protocol）によるコード補助機能です。  
デフォルトで `lua_ls`（Lua）と `pyright`（Python）がインストールされます。

| キー | 動作 |
|---|---|
| `gd` | 定義へジャンプ |
| `gr` | 参照箇所の一覧 |
| `K` | ホバードキュメントを表示 |
| `<leader>ca` | コードアクション（自動修正など） |
| `<leader>rn` | シンボルのリネーム |
| `<leader>lf` | コードフォーマット |

### 新しい LSP サーバーを追加する

`:Mason` コマンドを開いて `i` でインストールし、`plugins/init.lua` の `ensure_installed` に追加します。

```lua
ensure_installed = { "lua_ls", "pyright", "tsserver" },  -- TypeScript を追加
```

## 9. 補完 (nvim-cmp)

Insert モード中に自動的に補完候補が表示されます。

| キー | 動作 |
|---|---|
| `<C-n> / <C-p>` | 次 / 前の候補を選択 |
| `<Tab>` | 次の候補を選択 / スニペット展開 |
| `<CR>` | 候補を確定 |
| `<C-Space>` | 補完を手動で起動 |
| `<C-e>` | 補完を閉じる |

## 10. Git (gitsigns)

| キー | 動作 |
|---|---|
| `]c` | 次の変更箇所（hunk）へ |
| `[c` | 前の変更箇所（hunk）へ |
| `<leader>hs` | hunk をステージ |
| `<leader>hr` | hunk をリセット |
| `<leader>hp` | hunk のプレビュー |

---

## 11. コメント

| キー | 動作 |
|---|---|
| `gcc` | 行コメントをトグル |
| `gbc` | ブロックコメントをトグル |
| `gc` + モーション | 範囲をコメント（例: `gcip` で段落全体） |
| Visual モードで `gc` | 選択範囲をコメント |

---

## 12. プラグイン管理 (lazy.nvim)

| コマンド | 動作 |
|---|---|
| `:Lazy` | プラグイン管理画面を開く |
| `:Lazy update` | 全プラグインをアップデート |
| `:Lazy sync` | インストール・更新・クリーンを一括実行 |
| `:Lazy clean` | 不要なプラグインを削除 |

---

## Tips

- **which-key**: Normal モードで `<Space>` を押すと、利用可能なキーバインドの一覧がポップアップ表示されます。
- **`:checkhealth`**: Neovim の設定状態や LSP・プラグインの問題を診断できます。
- **`:TSInstall {言語}`**: Treesitter のパーサーを追加インストールできます（例: `:TSInstall go`）。
